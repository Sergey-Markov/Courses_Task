5000 и выше=> 50%
3000-4999 => 30%
1000-2999=> 10%
1-999 => 0%

Напишите программу, которая берет общую стоимость в качестве вводных данных и
выводит в консоль соответствующую скидку

Пример вводных данных:
4700
пример выходных данных:
50%

Подсказка: Используйте логический оператор && для объединения нескольких условий.
