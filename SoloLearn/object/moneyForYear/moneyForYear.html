Данная программа должна выщитать годовой доход по займу.

Она должна соответствующим образом использовать в качестве вводных данных сумму займа и годовую процентную ставку, а затем создать заем в качестве объекта. Завершите функцию вне конструктора для расчета годового дохода, после этого закрепите это значение за соответствующим полем конструктора( вам также необходимо его создать), для получения результата.

Пример вводных данных:

15000
20

Пример результата:
3000

Пояснение:
Результат - это годовой доход по займу, где первоначальная сумму равна 15000, а 20% - годовая процентная ставка: 20/100 * 15000 = 3000.

В конструкторе вы должны создать новое свойство для хранения возвращаемого из вычесленой функции значения дохода за год.
