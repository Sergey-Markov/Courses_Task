Директору магазина нужна программа для расчета скидки на товары.

Данная программа правильно создает объект товара с его номером и ценой в качестве свойств , но метод , который должен определять скидку, не завершен. Исправте его! Номер товара его цена и размер скидки принимаются в качестве вводных данных.

Пример вводных данных:

Лафет1493
1700
15

Пример результата:
Лафет1493 price:1700
Лафет1493 new price: 1445

Совет:

Первые вводные данные - это номер товара, вторые - цена без учета скидки, третьи - процент скидки.
Таким образом, с учетом скидки, новая цена будет составлять 1700-(0.15 * 1700)=1445
